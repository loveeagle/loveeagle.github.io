数组

  在程序中，我们经常需要使用大量数据，数组就是一个可以存储大量数据的基础结构。在数组声明、定义过程中，会在内存中开辟一块连续空间，用于存储数据。使用数组数据，是通过数组的下标（或称索引）来完成的，比如a[2] = 3;就是给数组 a 第2个元素赋值为3。

小鱼比可爱

题目分析

  本题中，小鱼可以看到它左边的其他小鱼的可爱程度，然后计算有多少只小鱼不如自己可爱。那么我们可以设置一个数组，数组的中元素代表小鱼的可爱程度。第 n 个小鱼看到的是第 0 ~ n-1 个小鱼的可爱程度，则计算有多少只小鱼不如自己可爱时，可以计算 第 0 ~ n-1 个元素是否比第 n 个小鱼小。设置第二个数组，将结果保存在数组中，最终再输出结果数组的内容。
